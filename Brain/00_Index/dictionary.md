# Dictionary

司令塔室で使う用語と分類です。

## Genre

| Genre | 意味 |
|---|---|
| `app-template` | 派生元になる原本アプリ |
| `project-app` | このプロジェクト専用に使うアプリ |
| `module-registry` | agent、skill、hook などの実行 module 管理 |
| `template-registry` | 原本テンプレートの管理 |
| `context` | カンバンや agent に渡す参照 context |
| `index` | 全体目次、辞書、パス登録 |
| `report` | agent の報告、調査結果、レビュー |
| `inbox` | 未分類の受信物 |
| `tech-stack` | 技術スタック、package、外部サービス |

## Kind

| Kind | 意味 |
|---|---|
| `app` | 実行可能なアプリ |
| `brain-folder` | Brain 配下の管理フォルダー |
| `agent` | 作業主体 |
| `skill` | 作業手順や能力 |
| `hook` | 起動条件や自動反応 |
| `document` | 説明・設計・学習資料 |
