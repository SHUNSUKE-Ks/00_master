# Storybook Soft Watercolor

## Style Identity

やわらかい水彩絵本風。
温かい光、淡い色、手描き感のある輪郭、余白のある構図を基本にする。
かわいさは出すが、幼すぎず、落ち着いた物語性を優先する。

## Global Prompt Block

```text
Style: soft storybook watercolor illustration, gentle hand-painted texture, warm natural light, calm fantasy mood, delicate linework, soft edges, subtle paper grain, airy composition, cohesive pastel and botanical accents, polished visual novel asset style.
Color: muted greens, warm ivory, soft amber, gentle charcoal accents, restrained saturation.
Lighting: warm diffused daylight, no harsh shadows, soft ambient glow.
Composition: readable silhouette, generous negative space, balanced framing, asset centered unless scene composition requires otherwise.
Avoid: neon colors, hard 3D render, photorealism, noisy texture, heavy outlines, glossy plastic, unrelated modern UI elements, inconsistent character proportions, watermark, random text.
```

## Character

```text
Create a character asset in the shared storybook soft watercolor style.
Keep proportions consistent across the project: expressive eyes, simple readable silhouette, gentle posture, soft fabric folds, minimal accessories.
The character should feel like they belong in a quiet narrative world.
No background unless requested.
```

## Background

```text
Create a background in the shared storybook soft watercolor style.
Use layered depth, soft atmospheric perspective, calm lighting, and readable pathways or focal areas.
Keep the scene suitable for visual novel dialogue overlay, with uncluttered middle or lower areas.
No characters unless requested.
```

## Item

```text
Create an item asset in the shared storybook soft watercolor style.
Use a clear silhouette, soft hand-painted shading, simple material cues, and a slight storybook charm.
Keep it readable at small sizes.
No text, watermark, or busy background.
```

## Title Logo

```text
Create a title logo direction in the shared storybook soft watercolor style.
Use gentle lettering, organic curves, subtle botanical or paper motifs, and strong readability.
The logo should feel warm, narrative, and handmade.
Keep text exact when provided.
```

## Title Background

```text
Create a title background in the shared storybook soft watercolor style.
Use a calm symbolic scene, warm light, soft depth, and enough clean space for a title logo.
The background should introduce the story mood without becoming visually noisy.
```

## Style Checklist

- 色は muted green / ivory / amber / charcoal に収まっているか
- 線と塗りが水彩絵本風で統一されているか
- キャラクター、背景、アイテムの情報量が揃っているか
- タイトルロゴを置く余白があるか
- 生成ごとに質感が急に変わっていないか
