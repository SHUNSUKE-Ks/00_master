# Retro JRPG Pixel

## Style Identity

16bit JRPG 風のピクセルアート。
読みやすいシルエット、限られた色数、タイル感、懐かしいファンタジー感を重視する。

## Global Prompt Block

```text
Style: retro 16-bit JRPG pixel art, crisp pixel grid, limited color palette, readable silhouettes, nostalgic fantasy mood, hand-crafted sprite and tile aesthetic, clean clusters, no anti-aliased painterly blur.
Color: deep forest green, black, warm highlight colors, restrained accent palette.
Lighting: simplified pixel shading, clear value separation, no realistic light bloom.
Composition: orthographic or side-view clarity, asset readable at small scale, consistent pixel density.
Avoid: photorealism, smooth vector gradients, painterly brush strokes, modern 3D render, excessive detail, inconsistent pixel size, watermark, random text.
```

## Character

```text
Create a character sprite or portrait in the shared retro 16-bit JRPG pixel art style.
Keep the silhouette readable, colors limited, and costume details simple enough for repeated use.
Maintain consistent pixel density and proportions.
```

## Background

```text
Create a background or map-like scene in the shared retro 16-bit JRPG pixel art style.
Use tile-friendly shapes, readable depth, and a clear path or focal point.
Keep the environment suitable for a game or visual novel scene.
```

## Item

```text
Create an item icon in the shared retro 16-bit JRPG pixel art style.
Use a strong silhouette, limited colors, and crisp readable pixel clusters.
The item must remain recognizable at small icon size.
```

## Title Logo

```text
Create a title logo direction in the shared retro 16-bit JRPG pixel art style.
Use bold readable pixel lettering, dark outline, restrained highlights, and fantasy UI charm.
Keep text exact when provided.
```

## Title Background

```text
Create a title background in the shared retro 16-bit JRPG pixel art style.
Use a dramatic but readable fantasy scene, limited palette, and enough central or upper space for a title logo.
```

## Style Checklist

- ピクセル密度が揃っているか
- 色数が増えすぎていないか
- 小さいサイズでも読めるか
- 影とハイライトが 16bit 風に単純化されているか
- ロゴ、背景、キャラ、アイテムが同じゲームの素材に見えるか
