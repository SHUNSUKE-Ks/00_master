# APP

司令塔室と連動するアプリを置く場所です。

## 現在のアプリ

- `Note/`  
  `note00-gallerynotedb_vol1.1` から移植したノートアプリです。
  汎用 Note の原本テンプレートとして保持します。

- `kanban-note01/`  
  `Note/` をベースに作成した、このプロジェクト専用ノートです。
  `kanban_June` と `Brain` に連動するカンバン特化ノートとして改造します。

- `novelEngine/`  
  `solidjs-novel-engine-01-short-schema` から移植した、ScenarioBox 専用の NovelEngine です。
  シナリオ JSON、背景、立ち絵、UI、音素材を使って短編ノベルを確認します。
