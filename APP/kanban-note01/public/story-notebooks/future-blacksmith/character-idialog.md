# 主人公

## 未来を打ち直す鍛冶屋

| 項目   | 内容                     |
| ------ | ------------------------ |
| 役割   | 主人公                   |
| 職業   | 辺境鍛冶屋               |
| 能力   | 打った武器から未来を見る |
| 特徴   | 戦士ではなく“未来の職人” |
| 欠点   | 全部背負い込む           |
| テーマ | 「戦わせない」           |

### ビジュアル案

- 炉火を見る癖
- 指に火傷跡
- 片目だけ未来視で疲弊
- 武器より工具が多い工房

### 内面

- 自分の剣が人を殺す感覚
- 道具を作る時だけ穏やか
- 「剣は未来を狭める」と感じている
