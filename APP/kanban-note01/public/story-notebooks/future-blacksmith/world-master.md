### あらすじ（なろう用）

> 辺境の鍛冶屋の息子には、作った武器から未来が見える。
>
> 幼い日、仲間たちのために木の剣を打った瞬間――炉の炎の向こうに映ったのは、戦場で血に染まった彼らの姿だった。
>
> 魔物が村を襲う。正規軍は来ない。幼馴染たちは剣を取り、義勇軍を結成する。建築を夢見た友が防壁を設計する。料理を愛した友が刃に慣れていく。
>
> 「この未来を戦場に奪わせたくない」
>
> 鍛冶屋は知っている。剣を打つほど戦いの未来が固まり、工具を打つ時だけ平和な未来が見えることを。
>
> 一度目の夜は、守れなかった。何度でも打ち直す――未来そのものを、鍛え直すために。

## 種族設定

### カエリス（守護一族）

- 星詠み族の巫女を守るために生まれた剣士の一族
- **固有能力**：剣を通じて使用者の未来を視る・技術を継承する
- この能力は他種族にほぼ知られていない
- 守護の手段として「剣による先読み」が一族の血に刻まれた

**敗戦後の扱い：**
- 過去に人間と戦い、敗北
- 以降、**剣の習得を禁止**される（反逆防止の名目）
- 実態：剣を奪うことで守護の力ごと封じるため
- 人間からは「亜人」と総称で呼ばれる（蔑称）

**主人公との関係：**
- 親がいなかったため、自分がカエリスだと知らずに育った
- 剣を握ると未来が見える理由も知らない
- Game 1の全行動が、無意識の「守護者」の本能だった

### 星詠み族

- 未来を「読む」巫女の一族
- 戦えない。見るだけ
- カエリスなしでは守られない存在
- 敗戦後の行方不明。生き残りが存在する（Game 2で登場）

### 世界の亜人構造

- 亜人族は複数種族が存在する総称
- カエリスはその一種族
- 人間社会に混在して生活しているが差別を受ける

## 能力・概念

### 未来視のルール

| 作るもの | 見えるもの |
|---------|-----------|
| 剣・武器 | 戦いの未来 |
| 工具・生活道具 | 平和な未来 |
| 未完成品 | 分岐・霧がかかった未来 |
| 折れた武器 | 死 |
| 打ち直し | 改変された未来 |

- **トリガー**：刃物に「名前・意志」を刻む瞬間
- 見えるのは**使用者の未来**（作った相手ごとに異なる）
- 見るたびに主人公の片目が疲弊する
- 本来の意味：巫女を守るために「先を読む」力だった

### スキル継承

- 武器には使用者の痕跡が残る（癖・思考・技術・感情・後悔）
- 剣を通じて未来の戦闘経験・型がインストールされる
- 道具ごとの継承例：
  - 包丁 → 精密性
  - 農具 → 土地感覚
  - 医療器具 → 人体知識

### 主人公の内面軸

> 「剣を打つほど、戦いの未来が固まっていく」

- 道具を作る時だけ、穏やかな未来が見える
- 自分の手が仲間を戦場に送っているという感覚
- 「戦わせない」が行動原理
- ※この苦しみの正体：守護の力が攻撃目的に使われているから

## 世界観

### 辺境と中央

| 場所 | 特徴 |
|------|------|
| 辺境 | 自衛文化。村同士の助け合い。小規模鍛冶屋が多い |
| 中央国家 | 魔物被害を軽視。正規軍は都市優先。辺境は自己責任。税だけ取る |

### 魔物

- 完全な悪ではない。**災害に近い存在**
- 季節移動・繁殖期・群れ行動・縄張り争いがある
- 種族例：犬型、熊型（他候補あり。非行型は禁止）
- **凶暴化の本当の原因**：魔素の淀み（Game 2で判明）

### 魔素の淀み（Game 2への伏線）

- 魔物が凶暴化する根本原因
- 星詠み族にしか感知・解決できない
- Game 1では「なぜ魔物がこんなに増えたのか」が未解決のまま終わる

## シリーズ展開

### Game 1 → Game 2 接続

```
【Game 1 エンディング】
  主人公、魔獣の大群を一人で撃退
  仲間は夢を生きる。主人公だけが消える
  主人公は最後まで「自分が何者か」を知らない
            ↓
  星詠みの巫女、現れる

【Game 2 開幕】
  巫女「魔獣が凶暴化した本当の原因は魔素の淀み」
  巫女「あなたはカエリス。私の守護者の血を引く者」
  主人公、初めて自分の正体を知る
  従者として巫女に同行。淀みの根源へ
```

### Game 1での行動がGame 2で意味を持つ

| Game 1での行動 | Game 2での意味 |
|---|---|
| 剣より道具を作りたかった | 守護者は戦士ではなく「盾」だった |
| 仲間を戦場に出したくなかった | カエリスの本能。守る者を傷つけさせない |
| 一人で全部背負った | 従者の在り方そのもの |
| 剣で未来が見えた | 巫女を守るための先読みの力だった |
| 魔獣を一人で撃退した | 守護者としての覚醒の瞬間 |
