TITLE GAME STUDIO / BEFORE ENGINE BINDING

2D_SRPGStudio

目的

GameLab WorkSpaceでは、ゲームEngineより先にTitleGameStudioを作り、 レイアウト、仕様、素材、画面Stateを決めてから novel / battle / collection Engine を当てる。

Engine候補

  • novel: 会話、ログ、選択肢
  • battle: SRPG map、行動、ターゲット選択
  • collection: 図鑑、報酬、実績

Gallery / Generated Assets

Package Slots / JSON Attach Space

DevStudioは単体HTMLの作業場であり、Gameに必要な情報はslotへJSONとしてattachする。 依存関係を満たしたslotからEngineへ渡す。
