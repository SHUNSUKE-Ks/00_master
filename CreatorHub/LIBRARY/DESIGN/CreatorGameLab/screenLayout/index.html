CreatorGameLab Screen Layout

4分割のコンセプト画像を、実装用のフルページ画面に分解します。

実装では4分割にしない

この画像は全体の雰囲気と情報設計の確認用です。実アプリでは Sidebar + Full Page View として各画面を広く表示します。

  • Title Select: フルページのタイトルギャラリー
  • Engine Sandbox: フルページの試験室一覧
  • Dev Save / Load: フルページの再開スロット一覧
  • Workspace: 後続開発用Placeholder
