# Launchers

WindowsのStartメニューへ追加するための起動bat置き場です。

| ファイル | 起動対象 | URL / 種別 |
|---|---|---|
| `start_kanban_june.bat` | Kanban_June | `http://127.0.0.1:5184` |
| `start_kanban_june_tauri.bat` | Kanban_June | Tauri desktop dev |
| `start_creator_game_lab.bat` | CreatorGameLab | `http://127.0.0.1:5194` |
| `start_kanban_note01.bat` | kanban-note01 | `http://127.0.0.1:5185` |

Startメニューに追加する場合は、このフォルダー内のbatへのショートカットを作成してください。
